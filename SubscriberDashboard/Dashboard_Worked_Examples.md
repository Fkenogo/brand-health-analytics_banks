## Example 1: Total Awareness Calculation

**Formula:**
```
Total Awareness = (Number aware ÷ Total sample) × 100
```

**Calculation:**
```
BK awareness = 720 ÷ 1,000 = 0.72
Total Awareness = 72%
```

**For all banks:**
```
BK: 720 ÷ 1,000 = 72%
I&M: 680 ÷ 1,000 = 68%
BPR: 750 ÷ 1,000 = 75%
```

**Ranking:**
1. BPR (75%)
2. BK (72%)
3. I&M (68%)

---

## Example 2: Top-of-Mind Awareness

**Formula:**
```
Top-of-Mind = (First mentions ÷ Total sample) × 100
```

**Calculation:**
```
BK Top-of-Mind = 150 ÷ 1,000 = 0.15
Top-of-Mind = 15%
```

**For all banks:**
```
BK: 150 ÷ 1,000 = 15%
I&M: 120 ÷ 1,000 = 12%
BPR: 180 ÷ 1,000 = 18%
Others: 100 ÷ 1,000 = 10%
Don't Know: 450 ÷ 1,000 = 45%
```

**Ranking:**
1. BPR (18%)
2. BK (15%)
3. I&M (12%)

## Example 3: Spontaneous Recall

**Formula:**
```
Spontaneous Recall = (Q1 mentions + Q2 mentions) ÷ Total sample × 100
```

**Important:** Don't double-count if someone mentioned in both Q1 and Q2

**Calculation for BK:**
```
Q1 (First mention): 150 people
Q2 (Other mentions): 280 people

Total Unique Mentions: 150 + 280 = 430 people
(Assuming no overlap - if overlap exists, subtract)

Spontaneous Recall = 430 ÷ 1,000 = 43%
```

## Example 8: Retention Rate

**Formula:**
```
Retention Rate = (Currently Using ÷ Ever Used) × 100
```

**Calculation:**
```
Currently Using: 288

```

**Interpretation:**

```

## Example 11: NPS Calculation

**Formula:**
```
NPS = % Promoters (9-10) - % Detractors (0-6)
```

**Step 1: Categorize**
```
Promoters (9-10): 45 + 72 = 117 people
Passives (7-8): 43 + 58 = 101 people
Detractors (0-6): 29 + 18 + 23 = 70 people

Total: 117 + 101 + 70 = 288 ✓
```

**Step 2: Calculate Percentages**
```
% Promoters = 117 ÷ 288 = 40.6%
% Passives = 101 ÷ 288 = 35.1%
% Detractors = 70 ÷ 288 = 24.3%

Check: 40.6% + 35.1% + 24.3% = 100% ✓
```

**Step 3: Calculate NPS**
```
NPS = 40.6% - 24.3% = +16.3

Rounded: NPS = +16
```

**Interpretation:**
NPS of +16 is below average for banking (typical: +30 to +50). Need improvement.

```

    (Committed × 100) + (Favors × 70) + (Potential × 40) +

# PART 5: BRAND MOMENTUM

## Example 15: Momentum Score

**Component Values:**

```
1. Awareness Growth:
   Previous period: 68%
   Current: 72%
   Growth: +4 pp (+5.9%)
   Normalized score: 65/100

2. Consideration Rate:
   Aware non-users: 720 - 288 = 432
   High intent (7-10): 180
   Rate: 180 ÷ 432 = 41.7%
   Score: 42/100

3. Conversion Rate:
   Ever Used ÷ Aware: 360 ÷ 720 = 50%
   Score: 50/100

4. Retention Rate:
   Current ÷ Ever: 288 ÷ 360 = 80%
   Score: 80/100

5. Adoption Rate:
   Preferred ÷ Current: 108 ÷ 288 = 37.5%
   Score: 38/100
```

**Weighted Momentum Score:**
```
(65 × 15%) + (42 × 25%) + (50 × 25%) + (80 × 20%) + (38 × 15%)
= 9.75 + 10.5 + 12.5 + 16.0 + 5.7
= 54.45

Momentum Score = 54/100
```

**Interpretation:**
Moderate momentum (54/100). Strong retention (80) but weak adoption (38). Focus on converting current users to primary status.

```
Changes: +2, -1, +2, +2, +2

```

**6-Month Growth:**
```

```

## Key Formulas Quick Reference

```
Awareness = Aware ÷ Sample × 100
Trial Rate = Ever Used ÷ Aware × 100
Retention = Current ÷ Ever Used × 100
Preference = Preferred ÷ Current × 100
NPS = % Promoters(9-10) - % Detractors(0-6)
Loyalty Index = Weighted sum of segments ÷ Total Aware
Momentum = Weighted(Awareness + Consider + Convert + Retain + Adopt)
Intent Avg = Sum of scores ÷ Respondents
Market Share = Your Preferred ÷ Total Sample × 100
MoM Growth = (Current - Previous) ÷ Previous × 100
Volatility = StdDev ÷ Mean × 100
```
