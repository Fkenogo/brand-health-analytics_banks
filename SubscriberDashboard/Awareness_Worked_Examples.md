# EXAMPLE 1: Total Awareness Calculation

## Raw Data from Questionnaire

**Section C, Q1 (Top-of-Mind):** "Which bank comes to mind FIRST?"
```
BK mentioned first: 150 people
Other banks mentioned: 400 people
Don't Know: 450 people
Total: 1,000 ✓
```

**Section C, Q2 (Spontaneous Recall):** "Which OTHER banks come to mind?"
```
BK mentioned (additional): 280 people
(These 280 did NOT mention BK first, but mentioned it in Q2)
```

**Section C, Q3 (Aided Awareness):** "Tick all banks you are aware of"
```
BK selected: 720 people
BK not selected: 280 people
Total: 1,000 ✓
```

## Step 1: Count Unique Aware Respondents

Logic: Anyone who mentioned BK in Q1 OR Q2 OR Q3 is "aware"

```
Method 1 (Using Q3 - Simplest):
Total Aware = 720 people

Method 2 (Combining Q1 + Q2 + Q3):
Q1: 150 people
Q2: 280 additional people (who didn't mention in Q1)
Q3 only: 290 additional people (aware but didn't recall spontaneously)
─────────────────
Total: 720 people ✓

Both methods match!
```

## Step 2: Calculate Total Awareness %

**Formula:**
```
Total Awareness = (Number aware ÷ Total sample) × 100
```

**Calculation:**
```
Total Awareness = 720 ÷ 1,000 × 100
                = 0.72 × 100
                = 72%
```

## Step 3: Interpret Result

**Your Result:** 72% awareness

**Benchmark Comparison:**
- 90-100%: Market leader ← Not here
- 70-89%: Strong established brand ← YOU ARE HERE ✓
- 50-69%: Moderate awareness
- 30-49%: Emerging brand
- 0-29%: New entrant

**Interpretation:** BK has strong awareness in Rwanda. 72% of banking-age population knows the bank.

# EXAMPLE 3: Spontaneous Recall

## Raw Data

**Q1 + Q2 Combined:**
```
Mentioned BK in Q1 (first): 150 people
Mentioned BK in Q2 (other): 280 people
Did not mention BK spontaneously: 570 people
──────────────────────────────────
Total: 1,000 ✓
```

## Step 1: Count Total Spontaneous Mentions

**Important:** Don't double-count!

```
Unique people who mentioned BK in Q1 OR Q2:
= 150 + 280
= 430 people
```

**Verification:**
```
Check: 430 mentioned + 570 didn't mention = 1,000 ✓
```

## Step 2: Calculate Spontaneous Recall %

**Formula:**
```
Spontaneous Recall = (Unprompted mentions ÷ Total sample) × 100
```

**Calculation:**
```
Spontaneous Recall = 430 ÷ 1,000 × 100
                   = 43%
```

## Step 3: Verify Relationship to Top-of-Mind

**Rule:** Spontaneous Recall must be ≥ Top-of-Mind

```
Spontaneous: 43%
Top-of-Mind: 15%

43% > 15% ✓ Correct!

This makes sense because:
Spontaneous = Top-of-Mind + Other unprompted mentions
43% = 15% (Q1) + 28% (Q2) ✓
```

# EXAMPLE 4: Awareness Quality Calculation

## Data from Previous Examples

```
Total Awareness: 72%
Top-of-Mind: 15%
```

## Calculation

**Formula:**
```
Awareness Quality = (Top-of-Mind ÷ Total Awareness) × 100
```

**Calculation:**
```
Awareness Quality = 15 ÷ 72 × 100
                  = 0.2083 × 100
                  = 20.83%

Rounded: 20.8% or 21%
```

## Interpretation

**Your Score:** 20.8%

**Quality Tiers:**
- 40%+: Excellent quality ← Not here
- 25-39%: Good quality ← Not here
- 15-24%: Moderate quality ← YOU ARE HERE ✓
- 10-14%: Weak quality
- <10%: Poor quality

**What This Means:**
Out of 720 people who know BK, only 150 (21%) think of it first. The other 79% know BK but don't prioritize it mentally.

**Implication:** BK has wide recognition but weak salience. Need campaigns to improve mental availability.

# EXAMPLE 5: Awareness Funnel Construction

## Step 1: Gather All Awareness Data

```
Total Sample: 1,000 people
Total Aware: 720 people (72%)
Spontaneous Recall: 430 people (43%)
Top-of-Mind: 150 people (15%)
```

## Step 2: Build the Funnel

```
TOTAL SAMPLE
    1,000 (100%)
        ↓
        │ 72% penetration
        ↓
TOTAL AWARENESS
    720 (72%)
        ↓
        │ 59.7% spontaneous conversion
        ↓
SPONTANEOUS RECALL
    430 (43% of sample, 59.7% of aware)
        ↓
        │ 34.9% top-of-mind conversion
        ↓
TOP-OF-MIND
    150 (15% of sample, 20.8% of aware, 34.9% of spontaneous)
```

## Step 3: Calculate Conversion Rates

**Stage 1 → 2 Conversion:**
```
Aware ÷ Sample = 720 ÷ 1,000 = 72%
```

**Stage 2 → 3 Conversion:**
```
Spontaneous ÷ Aware = 430 ÷ 720 = 59.7%
```

**Stage 3 → 4 Conversion:**
```
Top-of-Mind ÷ Spontaneous = 150 ÷ 430 = 34.9%
```

## Step 4: Calculate Drop-Offs

**Drop-off 1: Sample → Aware**
```
Not Aware = 1,000 - 720 = 280 people (28% lost)
```

**Drop-off 2: Aware → Spontaneous**
```
Aided Only = 720 - 430 = 290 people (40.3% of aware lost)
```

**Drop-off 3: Spontaneous → Top-of-Mind**
```
Other Mentions = 430 - 150 = 280 people (65.1% of spontaneous lost)
```

## Funnel Health Assessment

**Benchmark Conversions:**
- Aware → Spontaneous: Should be 50-70% ✓ (59.7% is good)
- Spontaneous → Top-of-Mind: Should be 25-45% ✓ (34.9% is good)

**Overall:** Healthy funnel with normal drop-offs at each stage.

# EXAMPLE 8: Month-over-Month Trend

## 6-Month Awareness Data

```
January: 68.0%
February: 70.0%
March: 69.0%
April: 71.0%
May: 73.0%
June: 75.0%
```

## Step 1: Calculate Each Month's Change

**Jan → Feb:**
```
Change (pp) = 70.0 - 68.0 = +2.0 pp
Change (%) = (2.0 ÷ 68.0) × 100 = +2.9%
```

**Feb → Mar:**
```
Change (pp) = 69.0 - 70.0 = -1.0 pp
Change (%) = (-1.0 ÷ 70.0) × 100 = -1.4%
```

**Mar → Apr:**
```
Change (pp) = 71.0 - 69.0 = +2.0 pp
Change (%) = (2.0 ÷ 69.0) × 100 = +2.9%
```

**Apr → May:**
```
Change (pp) = 73.0 - 71.0 = +2.0 pp
Change (%) = (2.0 ÷ 71.0) × 100 = +2.8%
```

**May → Jun:**
```
Change (pp) = 75.0 - 73.0 = +2.0 pp
Change (%) = (2.0 ÷ 73.0) × 100 = +2.7%
```

## Step 2: Summarize Trend

**Trend Table:**

| Month | Awareness | Change (pp) | Change (%) | Direction |
|-------|-----------|-------------|------------|-----------|
| Jan | 68.0% | - | - | Baseline |
| Feb | 70.0% | +2.0 | +2.9% | ↑ |
| Mar | 69.0% | -1.0 | -1.4% | ↓ |
| Apr | 71.0% | +2.0 | +2.9% | ↑ |
| May | 73.0% | +2.0 | +2.8% | ↑ |
| Jun | 75.0% | +2.0 | +2.7% | ↑ |

## Step 3: Calculate 6-Month Metrics

**Total 6-Month Change:**
```
Jun - Jan = 75.0 - 68.0 = +7.0 pp
Percentage: (7.0 ÷ 68.0) × 100 = +10.3%
```

**Average Monthly Change:**
```
Monthly changes: +2.0, -1.0, +2.0, +2.0, +2.0
Sum: +7.0 pp over 5 months
Average: +7.0 ÷ 5 = +1.4 pp per month
```

**Volatility (Standard Deviation):**
```
Values: 68, 70, 69, 71, 73, 75
Mean: 71.0
Deviations: -3, -1, -2, 0, +2, +4
Squared: 9, 1, 4, 0, 4, 16 = 34
Variance: 34 ÷ 6 = 5.67
StdDev: √5.67 = 2.38

Low volatility (<3) = Stable, consistent growth ✓
```

## Interpretation

**Strong positive trend:**
- +10.3% growth over 6 months
- +1.4 pp average monthly gain
- Low volatility (2.38) = consistent, predictable
- Only 1 decline in 6 months (March)

**Forecast:**
If trend continues, July awareness ≈ 75% + 1.4% = 76.4%

## Quick Reference: Key Formulas

```
Total Awareness = Aware ÷ Sample × 100

Top-of-Mind = First Mentions ÷ Sample × 100

Spontaneous Recall = (Q1 + Q2 unique mentions) ÷ Sample × 100

Awareness Quality = Top-of-Mind ÷ Total Awareness × 100

Share of Voice = Your Top-of-Mind ÷ All Top-of-Mind × 100

MoM Change (%) = (Current - Previous) ÷ Previous × 100

Demographic Awareness = Aware in segment ÷ Total in segment × 100
```
